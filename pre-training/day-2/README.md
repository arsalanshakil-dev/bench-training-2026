## What's the difference between a list and a dict

<!-- Describe the project: its purpose, core features, and who it's for. -->

Dict is a key-Value paired collection whereas list is an ordered collection which is based on index.

---

## When do you use each

<!-- Honest reflection on challenges, bugs, surprises, or things that took longer than expected. -->

List allow duplicates where as dict can not allow duplicate keys.

List is used to maintain order of items.
Dict is used to retrieve data quickly.

### Example

```python
# List Example
tasks = ["buy milk", "clean room", "code"]
print(tasks[1]) # Access by index: "clean room"

# Dictionary Example
user_data = {"name": "Alice", "age": 30}
print(user_data["name"]) # Access by key: "Alice"
```
